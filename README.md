# Key Features

  **1) User Authentication** 
  
       a) User registration and login
       b) JWT Token based Authentication
       c) Personalized experience after login

  **2) Places (Travel Module)**
  
       a) View a list of available travel places
       b) Select preferred places
       c) Track user-specific selected places
       d) Remove places from the selected list anytime
       
  **3) Shopping Module**
  
       a) Browse a list of products with prices
       b) Add items to cart
       c) Remove or update cart items
       d) Buy Now option for instant purchase
       e) Place Order directly from cart for selected products

  **4) Payment Integration**
       
       a) Secure payment processing using Razorpay
       b) Supports checkout from cart or Buy Now flow

  **5) AI Chat Assistant**
       
       a) Integrated Generative AI-based chat
       b) Helps users with queries and guidance
       c) Enhances overall user experience
